# README #

* Usage (commandline tool): 
```n64romdumpconverter.exe <path/to/rom_dump> <target_format>```
e.g. ```n64romdumpconverter.exe TLoZOoT.n64 v``` would convert the n64 file to a v64 file.
Possible target formats are n64 (n), v64 (v), z64 (z).

### What is this repository for? ###

* Tool for converting N64 ROM dumps (n64, z64, v64)

### How do I get set up? ###

* compile with Visual Studio 2022
* The already compiled executable is located in ./bin

### Who do I talk to? ###

* Repo owner or admin
